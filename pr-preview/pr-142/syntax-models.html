ArrheniusParameter

Define the nonlinear parameter as a function of temperature according to the Arrhenius law. The nonlinear parameter is therefore parametrized by the reference value and the activation energy.

activation_energy Activation energy in the Arrhenius law

  • Type: Scalar
ideal_gas_constant The ideal gas constant

  • Type: double
reference_value Reference value of the parameter

  • Type: Scalar
temperature Variable name for the temperature

  • Type: VariableName
  • Default: forces/T
