ArrheniusParameter

Defines the nonlinear parameter as a function of temperature according to the Arrhenius law. The nonlinear parameter is therefore parametrized by the reference value and the activation energy.

  • activation_energy
    • Description: activation energy in the Arrhenius law
    • Type: Scalar
  • automatic_scaling
    • Type: bool
    • Default: false
  • automatic_scaling_miter
    • Type: unsigned int
    • Default: 20
  • automatic_scaling_tol
    • Type: double
    • Default: 0.01
  • ideal_gas_constant
    • Description: the ideal gas constant
    • Type: double
  • reference_value
    • Description: reference value of the parameter
    • Type: Scalar
  • temperature
    • Description: variable name for the temperature
    • Type: LabeledAxisAccessor
    • Default: forces/T
  • use_AD_first_derivative
    • Type: bool
    • Default: false
  • use_AD_second_derivative
    • Type: bool
    • Default: false
