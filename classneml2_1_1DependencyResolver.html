Detailed Description

template<typename Node, typename ItemType>
class neml2::DependencyResolver< Node, ItemType >

The DependencyResolver identifies and resolves the dependencies among a set of objects derived from DependencyDefinition.

Template Parameters
NodeThe type of the node in the dependency graph, i.e. if this resolver is used to figure out dependencies among Models, this template parameter should be Model.
ItemTypeThe type of the consumed/provided items of each node
